Work in progress



## Installation
Using python3.10, run
1. `python3 -m venv venv`
2. `source venv/bin/activate`
3. `pip install -r requirements.txt`

If installation fails, try updating pip
1. `python3 -m pip install --upgrade pip`
2. `pip cache purge`
3. `pip install -r requirements.txt`
